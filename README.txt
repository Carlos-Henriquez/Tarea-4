

1. Una descripción general del sistema o proyecto:
- Este proyecto fue creado mediante la utilización NetBeans como IDE, JAVA como lenguaje y MySQL como base de datos.
- En este se elabora un un sistema de iniciar sesión donde gracias a la base de datos se verifica tanto el usuario y la contraseña dentro de estos para luego ir al listado de los usuarios.
- De escribir un usuario o contraseña correctos, saldrá un mediante una ventana emergente que verifique o se registre.
- En el apartado de registrarse al presionar el botón con el mismo nombre, deberá de llenar todos los campos de manera obligatoria ya que el programa sí reconoce que no los ha llenado, procede a notificarle mediante una ventana emergente cuales campos exactamente les falta por completar.
- Además de notificar los campos que faltan, notifica también a la hora de la contraseña y la verificación de la misma donde de no ser idénticas procede a notificar al usuario mediante una ventana emergente.
- Al ingresar de manera correcta un usuario y contraseña, procederá a la ventana de listado de usuarios donde se le mostrará un lista con todos los usuarios registrados dentro de la base de datos con sus respectivas informaciones.
- Además, dentro de esta ventana podrá realizar acciones como eliminar un usuario con tan solo hacer click en el usuario o la fila a la cual pertenece y luego presionando el botón de eliminar saldrá una ventana emergente notificando el éxito de la acción y volverá a la ventana de listado de usuarios. De no seleccionar un usuario y presionar el botón saldrá una ventana emergente notificandolo e instruyendo que hacer. También podrá actualizar los datos de un usuario que al igual que el botón mencionado anteriormente, debe de seleccionar el usuario y la columna a la cual pertenece y luego presionar el botón de actualizar, de lo contrario le notificará con las instrucciones pertinentes. De ser correcta la acción, se presentará la ventana de registro pero esta vez ayudará a realizar la actualización con el botón actualizar dentro de él, poseyendo las mismas características de lo que ocurre al no llenar todos los campos o no tener iguales los campos de usuario y contraseña, de ser exitosa la acción se notificará mediante una ventana emergente y volverá a la ventana de listado de usuarios. Si decide cerrar sesión, se cerrará la ventana de del listado de usuarios y volverá a la ventana del login.

2. El estado del proyecto:

3. Requisitos del entorno de desarrollo para la integración:
- Para la funcionalidad correcta de este programa se necesita el uso de MySQL y utilizar el script de la base de datos proporcionada en el folder "ScripBasededatos".
- Preferiblemente utilizar NetBeans como IDE para ejecutar el proyecto correctamente.
